# Stock Price Prediction

## How to Run
  `python3 stock_price_prediction.py`
  
  Then type in the ticker symbol (e.g. AAPL) you want to predict.
  
## Methods
We use Yahoo Finance API to fetch stock prices.

## Model
LSTM (Long Short Term Memory)
